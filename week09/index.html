    • Dialogs

  • window.confirm('') -- creates a confirm message.
  • window.prompt('') -- creates a pop up message with an input box
  • window.navigator.userAgent -- tells you what browser the user is using
  • window.location.href -- tells you what the url of the current page is.
  • window.location.protocal -- tells you is the url is secure or not
  • window.location.pathname -- returns the path name
  • windo.history.length -- returns how many times the user has visited that page
  • window.history.go() -- goes forward, reloads current page, or goes to the next page
  • window.open('') -- opens up new page in a popup
  • window.screen.height -- returns the height of the screen

    • Cookies

  • document.cookie = '' -- set cookie
  • document.cookie -- returns the value of the cookie
  • const cookies = document.cookie.split("; "); -- splits cookie into values
  • document.cookie = `name=Batman; expires=${ expiryDate.toUTCString()}`; -- set expiry date
  • document.cookie = 'name=Batman; secure'; -- make the cookie secure
  • document.cookie = 'name=Batman; expires=Thu, 01 Jan 1970 00:00:01 GMT'; -- delete a cookie
