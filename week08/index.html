Week 08

Remember this:

  • Transform property let's you move objects around the page.

    • Translation

  • similar position relative
  • transform: translate(45px, -45px);
  • -webkit-transform is for older browers
  • transforms don't work on inline elements
  • subsititute for margin-right or something

    • Scaling

  • transform: scale(1.5, 0.25);
  • makes an item larger
  • good for element:hover
  • transform: translateX(40px) scale(1.5);
  • don't declare a new transform because the first would override the second
  • does not effect the document flow

    • Rotation

  • roates the element by a certain degree
  • transform: rorate(10deg) translateX(40px) scale(1.5);

    • Skew

  • specefies a skew on the x and y axis.
  • transform: skew(15deg, 4deg);

    • Origin of the Transform

  • transform-origin: 0 0;
  • changes the point where the transform is rotating

    • Order of the Transform

  • the order matters, follow the steps

    • Watch for support for internet explorer 8 and earlier

    Transitions

    transitions give us the ability to create animations

    helps our pages feel more dynamic

  • transition-property: transform, color
  • duration: length of the animation
  • transition-timing-function: pace of the animation
  • cubic-bezier: same as the timing-function, just uses four parameters
  • transition-delay: allows the transition to wait before starting
  • combine them all together - transition: transform 0.2s ease-out 50ms;
  • transitionend event ends the transition

    • Animations

  • keyframe animations defines the start and end point
  • @-webkit-keyframes myAnimation { from{} to{}}-- add this for older browsers
  • animation-name: necessary for the animation to work, be careful that it contains a unique name
  • animation-duration: length of the entire animation
  • animation-timing-function: can be linear, ease, ease-in and more!
  • animation-iteration-count: how many times the animation runs
  • animaiton-direction: reverse or alternate, default is normal
  • animation-delay: how many milisections the browser should wait to start
